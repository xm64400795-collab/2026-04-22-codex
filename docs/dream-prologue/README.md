# 梦序章文档总览

这个目录汇总了《菌群觉醒》“梦序章”当前可执行的核心资料。

这些文档已经按使用对象分类，方便程序、美术、音频和策划分别查看，也方便后续补充版本迭代。

## 目录说明

### `flow`

- [梦序章_18节点关卡流程图_执行版.md](flow/梦序章_18节点关卡流程图_执行版.md)
  用途：定义 `N01-N18` 的关卡状态机、触发链、控制状态和情绪节奏。
- [梦序章_美术资源清单表_v1.md](flow/梦序章_美术资源清单表_v1.md)
  用途：把场景、角色、道具、特效拆成可排期、可分包的资源清单。

### `design`

- `梦序章详细执行策划_v2.docx`
  用途：完整章节策划说明，适合作为策划总稿。

### `program`

- `梦序章程序制作说明_易懂版_v2.docx`
  用途：给程序实现使用，适合继续细化成系统、状态机、触发和演出任务。

### `art`

- `梦序章美术制作说明_易懂版_v1.docx`
  用途：给美术执行使用，明确表现目标和限制。
- `梦序章美术AI提示词_中英双语_v1.docx`
  用途：给 AI 出图或外包沟通使用。

### `audio`

- `梦序章音频制作说明_易懂版_v1.docx`
  用途：给音频设计和制作使用。

## 推荐查看顺序

1. 先看 `flow/梦序章_18节点关卡流程图_执行版.md`
2. 再看 `flow/梦序章_美术资源清单表_v1.md`
3. 然后按职责进入 `design`、`program`、`art`、`audio`
